Monteur d'Ordinateur

Definition

Consiste à monter toutes les pièces d'ordinateur entre-elle pour constituer un ordinateur, généralement, après le montage, un système d'exploitation y est égamement installé par la suite

Les différentes Pièces :
  1. Processeur
  2. Carte Graphique
  3. Ram
  4. Carte Mère
  5. Alimentation
  6. Stockage
    • HDD
    • SSHD
    • SSD
